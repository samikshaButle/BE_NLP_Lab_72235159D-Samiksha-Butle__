# Assignment 2: Bag of Words, TF-IDF and Word2Vec

## Problem Statement
Implement text vectorization techniques using Bag of Words, TF-IDF and Word2Vec.

## Tools Used
- Python
- Jupyter Notebook
- scikit-learn
- gensim

## Description
**Bag of Words:** Converts text into frequency-based vectors.

**TF-IDF:** Assigns weights to words based on their frequency and importance across documents.

**Word2Vec:** Generates dense vector representations that capture semantic relationships between words.

## Conclusion
Text vectorization techniques help convert text data into numerical form, enabling machine learning models to process and analyze natural language effectively.
