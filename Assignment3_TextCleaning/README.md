# Assignment 3: Text Cleaning, Stopwords Removal, Stemming and Lemmatization

## Problem Statement
Implement text preprocessing techniques including text cleaning, stopwords removal, stemming and lemmatization using Python.

## Tools Used
- Python
- Jupyter Notebook
- NLTK

## Description
**Text Cleaning:** Removes punctuation and special characters from raw text.

**Stopwords Removal:** Eliminates commonly used words that do not contribute to meaning.

**Stemming:** Reduces words to their root form.

**Lemmatization:** Converts words into meaningful base (dictionary) forms.

## Conclusion
Text preprocessing techniques are essential for preparing textual data for Natural Language Processing applications.
